*ai-assistant.txt*   Neovim AI Assistant Plugin

A plugin to integrate AI-powered code assistance in Neovim.

==============================================================================
COMMANDS                                                  *ai-assistant-commands*

:AIModelSelector          Select an AI model using Telescope.
:AICommandPrompt          Run an AI command on selected code.

==============================================================================
KEYMAPS                                                    *ai-assistant-keymaps*

<Leader>am                Select AI model.
<Leader>ac                Run AI command on visual selection.

==============================================================================
CONFIGURATION                                            *ai-assistant-configuration*

require("ai-assistant").setup({
  provider = "openai",    -- Default provider
  model = "gpt-4o",       -- Default model
  keymaps = {
    select_model = "<leader>am",
    run_command = "<leader>ac",
  },
})
